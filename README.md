
# IMDb Clone

IMDb clone using HTML, CSS, and Javascript.

Link to production site: [IMDb clone](https://immortals430.github.io/IMDb-clone/)


## Features

- Search movie details by typing in the movie name in the search bar
- Add to watchlist
- Responsive web page
